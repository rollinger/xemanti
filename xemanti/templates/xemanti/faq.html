{% trans "Was ist Xemanti?" %}

Xemanti ist eine Datenbank zu der tausende Menschen täglich ihr Sprach-Wissen beitragen. Im Gegenzug können unsere Benutzer das Sprach-Wissen von vielen Millionen Beiträgen anderer Nutzer abfragen. Xemanti wächst also stetig - und jeder Beitrag macht Xemanti schlauer.

Andere Sprachtools bieten den Kommunikationsexperten eine feste Definition für ein Wort an. Das ist nützlich, aber Xemanti kann mehr! Xemanti will tiefer in die Sprache vorstoßen: Was ruft ein bestimmtes Wort in einem anderen Menschen hervor? Welche Wörter werden für Synonyme oder Antonyme gehalten? Wie erleben andere Menschen die Beziehungen zwischen Worten?

{% trans "Wie funktioniert Xemanti?" %}

Jeder Benutzer der Xemanti um sein Sprach-Wissen befragt, trägt sein eigenes Wissen zur Datenbank bei. Jeder Wissens-Beitrag wird in "Xetis" gutgeschrieben. Jede Wissens-Abfrage veringert den Betrag von Xetis.

Xemanti kennt in seinem frühen Stadium zwei Formen von Wissens-Beitrag: Assoziation und Sortieren. Assoziation: Der Benutzer bekommt ein Wort angezeigt und schreibt seine Assoziation zu diesem Wort in ein Textfeld und schickt es ab. Sortieren: Zu einem feststehenden Wort werden assoziierte Wörter gezeigt. Diese Wörter können dann per "Ziehen und Klicken" in Kategorien sortiert werden.

{% trans "Wie kann ich mitmachen?" %}

Natürlich kannst du ohne Anmeldung die Sprachtools von Xemanti ausprobieren. Klick auf diesen Link um direkt ein Wort deiner Wahl zu untersuchen.

Eine kurze Registrierung erleichtert dir das Arbeiten mit Xemanti und viele neue Funktionen werden in der Zukunft für dich freigeschaltet.
